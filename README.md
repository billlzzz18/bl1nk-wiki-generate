# Wiki AI Generator

Next.js + Tailwind v4 + Shadcn UI wiki with built-in AI image generator powered by DALL-E 3.

## Features
- Generate stickers, icons, emojis with custom prompts
- Real-time image preview (4 per generation)
- Copy markdown embed with one click
- Craft-style responsive grid layout
- Dark/light theme with blur glass cards
- Integrated views: markdown editor, kanban, calendar, table, chat
- Deploy instantly on Vercel

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS v4
- Shadcn UI components
- OpenAI DALL-E 3 API
- TypeScript
- React Query

## Quick Start
1. Clone repo
2. Add `OPENAI_API_KEY` in `.env.local`
3. `npm run dev`
4. Visit `/generator` to create AI images
